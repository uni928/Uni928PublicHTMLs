# Gmail Suspicion Meter

Gmailのメール一覧に「怪しさ%」を表示するChrome拡張です。AIへのプロンプト生成、ChatGPT/Geminiへの遷移、AI結果の貼り付け反映、圧縮記憶による次回以降の%表示に対応しています。

## 使い方

1. Chromeで `chrome://extensions/` を開きます。
2. 右上の「デベロッパーモード」をオンにします。
3. 「パッケージ化されていない拡張機能を読み込む」を押します。
4. このフォルダ `gmail-suspicion-extension` を選択します。
5. Gmailを開き、拡張機能アイコンを押すとパネルが表示されます。
6. 「ChatGPTで判定」または「Geminiで判定」を押します。
7. プロンプトがクリップボードにコピーされ、AI画面へ遷移します。
8. AIのJSON結果をパネル内の生成結果欄に貼り付け、「反映」を押します。

## ChatGPT遷移仕様

- プロンプトが9000文字以内の場合:
  - `https://chatgpt.com/?temporary-chat=true&prompt=XXX`
- 9000文字を超える場合:
  - `https://chatgpt.com/?temporary-chat=true`
- どちらの場合もプロンプトはクリップボードにコピーされます。

## 記憶する情報

メール本文そのものは保存しません。保存対象は次のような圧縮された特徴のみです。

- ドメイン別スコア
- 送信者別スコア
- 短いキーワードルール
- 件名・送信者に対する短い特徴ルール

## 注意

GmailのDOM構造に依存するため、Gmail側のUI変更でメール行の取得や表示位置がずれる可能性があります。
