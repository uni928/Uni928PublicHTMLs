🔐 Local only / no network access

Local HTML Locker

HTMLをブラウザ内で暗号化し、単体で開ける保護ページとして保存します。外部ライブラリや解析タグを使わず、公開前に扱いやすいシンプルな静的ファイルを生成します。

入力データは送信しません

ファイル読込、暗号化、出力ファイル作成はすべて現在のブラウザ内で処理します。

生成ページも自己完結

復号画面にも外部フォント・外部CSS・外部JavaScriptを含めません。

共有URLのキー対策

?key=... を使う場合は、読み取り後すぐにURLから削除します。

保護ファイルを作成

AES-GCM / PBKDF2
⌁
HTMLファイルをここへドロップ クリックして選択することもできます（.html / .htm）
公開用途では16文字以上の長い文字列を推奨します。自動設定した文字列はクリップボードにもコピーします。
?key=パスワード による自動解除を有効にする ページを開いた直後に値を読み取り、URLからkeyを消してから復号します。
容量を節約する 元HTMLの中身は変えずにgzip圧縮し、その圧縮データを暗号化します。復号時に解凍して表示します。
復号後HTMLを静的表示モードで開く 復号したHTMLから外部URL、スクリプト、イベント属性などを取り除いて表示します。動的なページは一部動かなくなります。

公開前チェック

この作成ページ自体には外部通信先を含めていません。CSPも通信禁止寄りに設定しています。

URLキー利用時

便利ですが、共有URLにパスワードが入る点はリスクです。短期共有や低リスク用途向けとして扱ってください。

静的表示モードの注意

安全寄りに表示するため、元HTML内のJavaScriptや外部画像などは削除されます。ページの動作維持を優先する場合はOFFにしてください。

容量節約

ONにすると、元HTMLをgzip圧縮してから暗号化します。元HTML自体は加工しない安全版相当の処理です。

禁止ワード

禁止ワードはこの作成ページのIndexedDBに保存されます。生成時に元HTML内をチェックし、一致した場合は処理を停止します。

補助ツールの候補

パスワードには120文字程度のランダムなBase62文字列を生成できる https://uni928.github.io/RandomStringGenerator/、アップロード前の容量削減には単一HTMLを自己解凍形式にできる https://uni928.github.io/Uni928PublicHTMLs/index53.html の利用も検討できます。

保存先

生成されたHTMLは通常の静的ファイルです。任意の静的ホスティングやローカル環境で利用できます。

Web Crypto APIを使用します。古いブラウザや安全でないコンテキストでは動作しない場合があります。
