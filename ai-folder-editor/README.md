# AI Folder Editor

フォルダを開いてテキストファイルを編集し、Alt + G でChatGPT用の反映パネルを開くローカルWebアプリです。

## 起動

```powershell
cd C:\Folder\ai-folder-editor
node server.mjs
```

ブラウザで `http://127.0.0.1:5177/` を開きます。

## 使い方

1. `フォルダを開く` で編集対象フォルダを選びます。
2. 左のツリーからテキストファイルを開きます。
3. `Alt + G` でChatGPT反映パネルを開きます。
4. 依頼プロンプトを書き、`プロンプトをコピーしてChatGPTを開く` を押します。
5. ChatGPTの出力パッチを結果欄に貼り付け、`反映する` を押します。
6. 表示された差分ごとに `提案を採用` または `元のまま` を選びます。

ChatGPTの出力は、通常はファイル全文ではなく `*** Old` / `*** New` の最小置換パッチになります。JSONパッチも読み取れます。
右の反映パネルで `中央パネルで確認` を押すと、選んだ差分が中央ペインのコード表示内に赤/緑で表示されます。
