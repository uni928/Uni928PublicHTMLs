CSS セレクタ統合抽出ツール

HTML と複数 CSS を読み込み、指定した class / id の最終的なCSSを1つにまとめて出力します。

HTML 内の <style> と <link rel="stylesheet"> の順序も見ます。
HTML の link 順に名前が一致するCSSを配置し、未参照CSSは選択順で末尾に追加します。
基本は .btn-72 のように入力してください。複数セレクタはカンマ区切りで入力できます。

設定

※ 疑似クラス（:hover等）は通常状態ではマッチしないため、厳密モードでは除外されます。厳密モードOFFでは、対象文字列を含むルールも候補にします。
